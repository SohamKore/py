Python Lists - Notes

What is a List?

Lists are used to store multiple items in a single variable. They are one of the four built-in data types in Python used to store collections of data.

Lists are created using square brackets []:

mylist = ["apple", "banana", "cherry"]
List Properties
  • Ordered: Items have a defined order, and the order will not change.
  • Changeable: You can change, add, or remove items in a list.
  • Allow Duplicates: Lists can have duplicate values.
List Length

Use the len() function to determine the number of items in a list.

print(len(mylist))
List Data Types

List items can be of any data type:

list1 = ["apple", "banana", "cherry"]
list2 = [1, 5, 7, 9, 3]
list3 = [True, False, False]
The list() Constructor

You can also use the list() constructor to create a list:

thislist = list(("apple", "banana", "cherry"))
Python Collections

There are four collection data types in Python:

  • List: Ordered, changeable, allows duplicates.
  • Tuple: Ordered, unchangeable, allows duplicates.
  • Set: Unordered, unchangeable, no duplicates.
  • Dictionary: Ordered, changeable, no duplicates.
